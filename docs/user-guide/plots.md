# Reading The Plots

## Feature UMAPs

Feature UMAPs show where a gene or mapped feature is expressed across the embedding. They are best for seeing expression patterns across transcriptional neighborhoods.

## Violin Plots

Violin plots summarize expression distributions by group. They are useful for comparing broad cluster or condition-level patterns but can compress sparse expression.

## Averaged Expression By Cluster

The averaged-expression heatmap shows mean normalized expression per cluster/group. It is designed to replace cell-level heatmaps when the goal is cluster-level comparison.

Use this plot to answer questions such as:

- Which clusters have the strongest average expression?
- Are several genes co-enriched in the same cluster?
- Does expression appear cluster-specific or broadly distributed?

## Dot Plot

The dot plot combines average expression and percent of cells expressing each gene. It is usually the most compact plot for comparing multiple genes across many clusters.

## Ridge Plots

Ridge plots show expression distributions across groups. They are most useful for single genes or small gene panels.

## Downloads

Each plot card includes download controls. Downloads include atlas provenance where supported, so keep the downloaded file together with the atlas version/date for reproducibility.
