# Exports And Citation

## Figure Downloads

Plot cards include download buttons. Use these for figures, presentations, or supplementary exploration records.

Recommended practice:

- Keep the downloaded figure with atlas version and date.
- Record species, integration, grouping, selected genes, and source species for cross-species plots.
- Use the ortholog trace download when interpreting cross-species mappings.

## Tables

Marker tables and mapping tables can be downloaded when available. These are useful for auditing which features were plotted and why some genes were omitted.

## Citation

Cite the atlas version used for your analysis and the original datasets and computational methods that support the specific results you interpret.

> Pereira W. et al. A cross-species single-cell atlas of legume root nodule symbiosis. Legume Root Nodule Symbiosis Atlas, version 1.0.
